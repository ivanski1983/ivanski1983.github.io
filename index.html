Customize your background:

To use your own background image, add the following CSS to the <style> tag at the top of this file and replace the URL with your image link:

/* Paste this inside the <style> tag at the top of the file */
body {
    background-image: url('https://your-image-url.com/path-to-your-image.jpg');
    background-size: cover;
    background-position: center;
    background-repeat: no-repeat;
    background-attachment: fixed;
    /* Optional: Add a semi-transparent overlay to keep text readable */
    position: relative;
}
/* You can also add a color overlay */
body::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.5); /* 50% black overlay */
}
